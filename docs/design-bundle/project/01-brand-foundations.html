/ 01.1

Logo usage The wordmark and gradient star are the only brand elements that don't change between directions. The rest of the page is theming on top of these.

Anatomy

Gradient mark
cyan → violet → magenta Wordmark
"Inturn" custom

Wordmark + gradient star always lock-up together at standard sizes. Minimum size: 20px tall on screen, 14mm in print. Below that, use the standalone mark.

Standard sizes

18px · favicon, dense nav
24px · sidebar, app header
32px · auth screens, modals
48px · marketing hero, splash

Clear space

x x

Minimum clear space on all sides equals the height of the gradient star (x). Never crop, lock against an edge, or overlap with adjacent UI.

On color · what works

White · default
Cream · marketing
Ink · needs inverted wordmark
Gradient · only with white wordmark

Never

Don't desaturate the gradient star.
Don't stretch or skew.
Don't recolor the gradient.
Don't place on low-contrast pinks/purples.
